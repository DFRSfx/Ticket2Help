Detailed Description

Implementação do padrão Singleton para configurações do sistema.

Garante que existe apenas uma instância das configurações em toda a aplicação.

Padrão Utilizado: Singleton Objectivo: Controlar acesso a configurações globais Benefícios:

  • Controlo de acesso
  • Ponto único de configuração
  • Thread-safe
